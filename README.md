# Shrike

Ruby RBAC(Role Based Access Control) System for ActiveRecord

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'shrike'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install shrike
